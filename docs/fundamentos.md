## 📌 ¿Qué es programar?

Programar es darle instrucciones a la computadora.

---

## 🧠 Conceptos básicos

- Variables → guardar datos
- Funciones → ejecutar acciones
- Condiciones → decisiones
- Bucles → repetir acciones
