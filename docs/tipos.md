# Tipos de datos en JavaScript

---

## 📌 Tipos primitivos

String: "Hola"
Number: 10
Boolean: true
Undefined: let x;
Null: null

---

## 📌 typeof

typeof "hola"
typeof 10

---

## 🔥 Conversión

Number("10")
String(10)
Boolean(1)

---

## ⚡ Comparaciones

10 == "10"   // true
10 === "10"  // false

---

## 🧠 Truthy / Falsy

0 → false
"" → false
null → false
undefined → false
1 → true

---

## 🧠 Resumen

JavaScript tiene tipos dinámicos
